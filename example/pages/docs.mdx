# Overview

## What is it?

Gaia is a thin layer that simplifies the development of web applications, servers and automations. You can build almost anything you want and have it in production effortlessly.

## How is it built?

One of the goals of this project is having the smallest possible core and being able to build itself from that core, the same way a user would do for building their apps.
